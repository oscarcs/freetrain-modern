                               FreeTrainEX-Aver

=========================================================================
                                              by FreeTrainEx-Aver Project
                         (http://sourceforge.jp/projects/freetrain-exav/)

FreeTrainは、ARTDINK社の鉄道会社経営シミュレーションゲーム「A列車で行こう
」のクローンを目指したゲームです。
残念なことに2005年にFreeTrainExが公開されたのが最後で、その後に開発は停止
してしまいました。

そこでFreeTrainExの最終バージョンを引き継ぎ、更に開発を続けていこうとして
いるのが当プロジェクト「FreeTrainEx-Aver Project」です。
当プロジェクトはあくまで１つの機能実装例であり、FreeTrainの正当な後継では
ありません。(開発者も異なります)
当プロジェクトの最終的な目標は鉄道ファンにも経営ゲームファンにも納得して
頂けるものを公開することです。

◎動作環境
  対応ＯＳ
    Windows 2000
    Windows XP
    Windows Server 2003 ※
    Windows Vista ※
    Windows Server 2008 ※
    Windows 7 ※
　　（※のものでは、OS単体＋FreeTrainEX Av単体では動作しません。DLLのインストール
　　　　などが必要となりますが、FreeTrainEX-Aver Projectではサポートしません）
    Windows 7

  必要ハードウェア
    16bit以上のcolor depthをサポートしているビデオカード

  必要ソフトウェア
    Microsoft Direct X 8.0以降(Direct X 10を除く)
    Microsoft .Net Framework 2.0以降
    (FreeTrainEx-Averから2.0に変わりました)

このゲームは、2ch「Ａ列車で行こう」スレッドの皆さんと一緒に作ったゲームで
す。477さん、暇さん、915さん、絵師見習いさん、旅行の友さん、鐘声さん、
後援者Ａさん、ミハさん、imaさん、Minさん、綾瀬さんを始めとして、画像や
音楽や意見を出してくれた皆さん方に感謝します。
そして当プロジェクトも2ch「鉄道経営系ゲーム制作スレッド」の皆さんと協力し
